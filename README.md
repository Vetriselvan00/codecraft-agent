# CodeCraft Agent – Ecommerce Product Analysis

CodeCraft Agent is a simple multi-agent pipeline that:

✔ Scrapes ecommerce products from a public test site  
✔ Analyzes price, rating, and product text  
✔ Generates sentiment & specification scores  
✔ Ranks products with a weighted scoring model  
✔ Produces a clean recommendation report

This project uses **requests, BeautifulSoup, pandas**, and simple Python logic.
